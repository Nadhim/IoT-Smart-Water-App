# IoT-Smart-Water-App


## Installation

- Clone the repository

```bash
    git clone https://github.com/Nadhim/IoT-Smart-Water-App.git
    cd ./IoT-Smart-Water-App
```


- Set up a virtual environment and install required packages

```bash
    python -m venv venv
    pip install -r ./requirements.txt
```
